# Third-party icons

Four vector drawables under `app/src/main/res/drawable/` are redrawn from
Google's [Material Symbols / Material Icons](https://github.com/google/material-design-icons)
as flat `pathData` (no image files, no code copied beyond the path shapes):

- `ic_delete.xml` -- "delete"
- `ic_download.xml` -- "file_download"
- `ic_edit.xml` -- "edit"
- `ic_settings.xml` -- "settings"
