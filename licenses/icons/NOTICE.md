# Third-party icons

Four vector drawables under `app/src/main/res/drawable/` (each existing as
three theme-suffixed copies -- `_ge`/`_ne`/`_adaptive`, see the main README)
are redrawn from Google's
[Material Symbols / Material Icons](https://github.com/google/material-design-icons)
as flat `pathData` (no image files, no code copied beyond the path shapes):

- `ic_delete_*.xml` -- "delete"
- `ic_download_*.xml` -- "file_download"
- `ic_edit_*.xml` -- "edit"
- `ic_settings_*.xml` -- "settings"
